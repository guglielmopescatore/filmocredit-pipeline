📁 VIDEO FILES DIRECTORY
========================

🎬 Place your video files here for processing with FilmoCredit.

📝 Instructions:
   1. Copy your video files (.mp4, .avi, .mkv, etc.) to this folder
   2. Run FilmoCredit and select your video from the file browser
   3. The processed results will appear in the adjacent 'episodes' folder

✅ Supported formats: MP4, AVI, MKV, MOV, and other common video formats

📞 Need help? Check the main README.txt in the parent directory.
